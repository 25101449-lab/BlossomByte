üå∏ COMPUTER HARDWARE üå∏

A general-purpose electronic counting device used in data processing due to its accuracy and high speed is called a computer. It is composed of two main components: hardware and software. The hardware refers to the physical components of a computer which includes the central processing unit and peripheral equipment for data input, output, and storage. Input devices, such as a keyboard and mouse, allow users to enter data, while output devices, like monitors and printers, display or produce information (Frankenfeld, 1993).

Tangible items that can be touched or seen are called hardware components where it is required to be able for a computer to function. There are two classifications of hardware: external and internal. This refers to the placement of physical parts around the device that is responsible for the functionality and use of computers. External includes keyboards, keyboard cases, etc., while internal includes video card, memory, or central processing unit (Denomme & Zandbergen, 2023).

Components that are fitted together inside a computer case to perform a task are called hardware components where it can be touched and seen. It includes lots of parts that perform their own functions that helps with the overall functioning of a computer. It includes: Motherboard, Central processing unit (CPU), Random access memory (RAM), Storage drive (HDD or SSD), Graphics processing unit (GPU), Keyboard, mouse, and monitors (Crucial, 2025).

The collective term ‚Äúcomputer hardware‚Äù is used to describe any parts that are physical of an analog or computer. While software consists of written, machine-readable programs, the hardware accepts the instructions and executes them. Hardware is categorized into two: internal and external where internal components are necessary for the proper functioning of the computer, while external components are parts that are attached to the computer and add functionality (Awati & Rosencrance, 2021).

Computer hardware extends far beyond traditional desktop computers. It now also powers everything such as enterprise servers, cloud data, smartphones, vehicles, smart medical devices, and other technologies. People stay connected and businesses operate due to the widespread integration of computer hardware. It is the backbone of operations which allows efficient data flow and drives innovation (Sunsjara & Smalley, 2025).

üå∏ SOFTWARE üå∏

In a computer system, the software is basically a set of instructions or commands that tell a computer to do a specific task that serves its users. It makes the hardware work as per user requirements by giving instructions. Examples of software are MS-Word, MS-Excel, PowerPoint and Web Browser. It is composed of two different types: system software and application software. System software is a language processor and device driver, while application software gives special functions or provides functions (GeeksforGeeks, 2025).

üå∏ System Software

A program that is designed to run a computer‚Äôs hardware and applications that also manages resources, memories, processors, and devices is called a system software. It provides a platform that runs application softwares and other softwares that are typically bundled with a computer‚Äôs operating system. There are different types of system software: operating system, device drivers, firmware, basic input/output system and unified extensible firmware interface, and programming language translator (Kitthu, 2025).

A software that provides a platform for application software to function efficiently and manage and coordinate operations of computer hardware is called system software. It enables the connection and control of components such as the CPU, RAM, hard drives, and other devices, and it is considered as the ‚Äúheart‚Äù of every computer system. Its key functions are: manage system resources, provide an operating environment for application software, control and monitor system activities, ensure system security and protection, optimize system performance, and etc. (Trang, 2025).

üå∏ Application Software

A software that aids end users and perform specific functions that deals with a direct interaction is called an application software. It lets users do specific tasks that involve in applications such as web browsers, microsoft word, and excel, which are examples of application software that can be used or accessed through computers. It can also be found in smartphones such as, whatsapp, telegram, games like candy crush, and ludo. Weather and transportation information can also be found in apps that can help people in their everyday lives as they rely on it (Satish, 2025).

Application software consists of specialized programs designed to help users perform specific tasks on their computers. Unlike system software, which manages hardware and basic operations, application software focuses on day-to-day user activities. It turns computing power into practical tools for problem-solving, content creation, and communication, offering benefits such as improved productivity, cost efficiency, accuracy, customization, collaboration, and accessibility (VirtuBox, 2025).
